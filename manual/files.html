Files

Files হল এ ফ্রেমওয়ার্কের কোর লাইব্রেরী। এটি File সংক্রান্ত যাবতীয় কাজ সম্পাদন করে। এ সংক্রান্ত সকল কাজ Files instance ব্যবহার করে করতে হবে।

Download File

download() খুবই প্রয়োজনীয় একটি মেথড যার মাধ্যমে খুব সহজেই আপনি যেকোনো File download এর কাজ করতে পারবেন। এটি ডাইনামিক ডাউনলোড সিস্টেম। ইউজার কখনওই আপনার ডাউনলোডকৃত ফাইলের লোকেশন জানতে পারবেনা। আপনি চাইলে আপনার সার্ভারের ফাইলেকে অন্য নামেও ডাউনলোড করাতে পারেবেন। এ মেথডটির দুটি প্যারামিটার একটি ফাইলের পুর্নাঙ্গ পাথ আর অন্যটি আপনি কি নামে ফাইলটি ডাউনলোড করাবেন সে নামটি
ধরি assets/doanload/abc.jpg file টি ডাউনলোড করবেন। এবং যখন ডাউনলোড হবে তখন এর নাম পরিবর্তন হয়ে হবে my_photo.jpg 
Files::download(basePath()."assets".DS."download".DS."abc.jpg", "my_photo.jpg");
এর ফলে abc.jpg file টি my_photo.jpg নামে ডাউনলোড হবে।
